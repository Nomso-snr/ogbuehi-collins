Binary System

Binary is a base-2 numeral system used by computers. It uses only two digits: 0 and 1.

  • Each digit is called a bit
  • 8 bits make a byte
  • Used in all digital systems and computing
  • Easy to implement with electronic circuits (on/off states)
Fun fact: Binary was first described by Gottfried Leibniz in the 17th century.
