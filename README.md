
# Decimal to Binary Converter

A responsive web application that converts decimal numbers (including fractions) to binary representation in real-time.

## Features

- *Real-time Conversion*: Binary result updates as you type
- *Fraction Support*: Handles decimal numbers with fractional parts
- *Responsive Design*: Works on desktop, tablet, and mobile devices
- *Visual Feedback*: Input validation with color coding
- *Sample Numbers*: Quick examples to try
- *Conversion Details*: Shows step-by-step conversion process
- *Keyboard Shortcuts*: 
  - Ctrl+K: Focus on input field
  - Enter: Convert
  - Escape: Clear input

## How to Use

1. Enter a decimal number in the input field (e.g., 15.625)
2. The binary equivalent will appear automatically
3. Click "Try Sample" for example conversions
4. Click "Clear" to reset the converter
5. Click on any example in the "Quick Examples" section to try it

## Examples

- 10 → 1010
- 27.75 → 11011.11
- 3.14159 → 11.001001000011...
- 255 → 11111111
